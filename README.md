# groupomania-router

## Project setup
```
open terminal => cd front
npm install
open cmd terminal => cd back
npm install
```

### Compiles and hot-reloads for development
```
on cd front =>
npm run serve
on cd back => 
nodemon server

*** RUN BOTH COMMANDS AT THE SAME TIME IN TWO DIFFERENTS TERMINALS ***
```
